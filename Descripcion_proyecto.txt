 
Descripción del proyecto
App para mostrar al público el trabajo que hicimos, desde las grabaciones de las sesiones, mostrar las carpetas que han creado los integrantes, mostrando casos de uso, descripción, equipos en los que estuvimos, será una app que se desarrollará el fluter y supabase, el backend será con Java en springle, crear API para crear conexión entre backend y fluter, vamos a poder ver los equipos que existen y los agentes que se crearon, y mostrar toda la información del agente y la información de quien la desarrollo, (desempeñó, asistencia, y avances que esa persona lleva en desarrollo del agente)
La aplicación se construirá con Flutter y Supabase, mientras que el backend se desarrollará en Java utilizando Spring. Se implementará una API para permitir la comunicación entre el backen
